# Project Overview

## Objective

The objective of this project is to design and implement an end-to-end Security Operations Center (SOC) monitoring solution capable of detecting SSH brute-force attacks against Linux servers using Splunk Enterprise.

The lab demonstrates the complete security monitoring lifecycle, including infrastructure deployment, attack simulation, log collection, detection engineering, dashboard development, and automated alerting.

---

## Project Scope

The project includes:

• AWS Infrastructure

• Ubuntu Linux

• Kali Linux

• Splunk Enterprise

• Splunk Universal Forwarder

• SSH Authentication Monitoring

• Hydra Attack Simulation

• Detection Engineering

• Dashboard Development

• Scheduled Alerting

---

## End Goal

Provide SOC analysts with visibility into SSH authentication attacks while demonstrating practical SIEM engineering skills.
